RainfallCalculator v1.0
10 August 2016
by Sean Coffey - 220160725

Python program to calculate average weekly and daily rainfall.

Written for Python 3.x - will have limited functionality for python 2.x.
Tested in Python 3.5.2.

Usage:

	Run program as normal python scripts

		python RainfallCalculator.py
	
	Program will prompt you to enter number of weeks over which you want to calculate rainfall averages.
		This value MUST be an integer greater than zero.
	
	The program will then ask you to enter rainfall values in millimetres for each day (individually) over the number of weeks entered. 
		Rainfall values MUST be non-negative integers. 

	Program will then return the total rainfall and the average rainfall per week and day.
